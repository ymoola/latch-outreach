Application Guide
App UI
Cloud canvas with Haze cards. Slate text. Interactive blue for CTAs only — one saturated color in the whole interface. Feels like a premium iOS app.
Marketing / Social
Haze and Fog backgrounds. Slate headlines. No loud colors. The quietness is the differentiator — every competitor is screaming, you whisper.
Investor / Print
White ground, Slate for body, Interactive blue for charts and key metrics. Clean enough to print on premium stock without looking washed out.
